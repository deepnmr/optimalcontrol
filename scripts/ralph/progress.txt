## Codebase Patterns
- Build backend: use setuptools (hatchling not available in this environment). Set `[tool.setuptools.packages.find]` with `include = ["optimalcontrol*"]` to avoid setuptools detecting refdoc/ as a top-level package.
- Typecheck: run `mypy optimalcontrol/` from project root; requires at least one .py file in the directory.
- Install: `pip install -e ".[dev]"` installs runtime + dev dependencies in editable mode.
- Tests: use `python3.10 -m pytest` (or `python3 -m pytest`), NOT bare `pytest` — the pytest script fails to pick up the editable install path in this environment. The `-m` flag ensures sys.path includes site-packages and the editable finder is registered.
